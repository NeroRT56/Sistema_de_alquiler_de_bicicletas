# Bike-Rental-System
Un sistema completo de alquiler de bicicletas implementado en Python usando programación orientada a objetos.

# El sistema de alquiler de bicicleta tiene 4 funcionalidades 
- Crear un usuario
- Registrar Bicicleta 
- Elimiar Bibicletas 
- Rentar Bicicletas 
- devolver biclcleta

## Los clientes pueden
- Ver bicicletas disponibles en la tienda.
- Alquiler de bicicletas por
- Generar factura 

### Los administradores pueden
- Agregar Bicicletas 
- Encontrar Bibicletas
- Cobrar al Cliente 
- Encontrar Usuario 
- colocar el precio por hora 

Recomendaciones 
•	Primero cree un usuario para poder saber a quien se le renta la bicicleta
•	Segundo registre una bicicleta para poder rentar una bicicleta 
•	En la opción 4 para rentar una bicicleta debe elegir el primer número que aparece de izquierda a derecha. Lo mismo para el usuario
